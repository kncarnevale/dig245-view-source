How to View Source Code of a Web Page on Safari

"Deep Dive"

  1. Right click on empty part of web page.
  2. Click "Inspect".
  3. Now you should be able to see the source code.

More Cool Things Dev Tools Can Do!

  • Allows you to update your web page on the browser source code, so you can see real time updates.
  • Can test Javascript code in console
  • Can use console.log() statements to print out things and check status of code and even send secret messages to other developers.
